# CIC0235-TR1
Trabalho da matéria de Teleinformática e Redes 1 na UnB. Implementa uma simulação de camada física e camada de enlace de um sistema de comunicação.

Para o funcionamento do programa, verifique se:
1. Python 3 está instalado
2. GTK está instalado
3. Use pip para instalar PyGObject, numpy e matplotlib
4. Use o comando `python InterfaceGrafica.py` para rodar o programa.
